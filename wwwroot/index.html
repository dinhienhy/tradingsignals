Example Usage
Webhook Endpoint Format
POST /webhook/{path}
JSON Payload Structure
{
  "secret": "your-webhook-secret",
  "symbol": "EURUSD",
  "action": "BUY",
  "price": 1.05432,
  "timestamp": "2025-08-09T10:30:00Z",
  "message": "Optional message about the signal"
}
Testing with curl
curl -X POST http://your-host/webhook/your-path \
  -H "Content-Type: application/json" \
  -d '{"secret":"your-secret","symbol":"EURUSD","action":"BUY","price":1.05432,"timestamp":"2025-08-09T10:30:00Z"}'
